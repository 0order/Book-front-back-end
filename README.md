# Book-front-back-end

## About
 A simple website project to catalog books, including information such as: Name, Author's Name and Genre, list all the information previously sent by the user and search for specific books among them.

---

## What is needed
• To run this project, you need to set your own user and password at AcessoBanco/Conexao.cs

• The database server used is MySql Workbench. In the GerenciamentoLivros folder you will find a .sql file with all the specifics to create and drop the tables needed
